Place the following font files in this folder to enable local fonts used by the app:

Required files (WOFF2 format recommended):
- GreatVibes-Regular.woff2
- Merriweather-Regular.woff2
- Merriweather-Bold.woff2
- Roboto-Light.woff2
- Roboto-Regular.woff2
- Roboto-Medium.woff2
- Roboto-Bold.woff2
- Pacifico-Regular.woff2

Notes:
- These filenames must match exactly, as referenced by /fonts/<file>.woff2 in src/index.css.
- You can generate .woff2 files from TTF/OTF with tools like fonttools or online converters.
- Ensure you have the right to redistribute the fonts you include.
